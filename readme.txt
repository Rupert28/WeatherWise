Display an hourly forecast for the next 24 hours on your WordPress website using OpenWeather's API.

== Description ==

The WeatherWise plugin provides a simple way to display a 24-hour weather forecast for your location using data retrieved from OpenWeather's API.

== Installation ==

1. Go to `Plugins` > `Add New` in your WordPress Dashboard.
2. Search for `WeatherWise`.
3. Click `Install Now` and then `Activate`.

== Configuration ==

Navigate to `WeatherWise` in the WordPress admin menu to configure the plugin settings.

- **Weather API Key**: Add your API key for weather data retrieval. To access the hourly forecast, subscribe to OpenWeather's One Call API 3.0. Once subscribed, enter your API key in the settings.
- **Customization**: Customize the location and display name of the widget using the options in the admin menu.

== Usage ==

To display the weather forecast on your site:

- Use the provided shortcode `[display_weather]` within your pages or posts.

== Features ==

== Frequently Asked Questions ==

### How do I update the weather information?

Weather information updates automatically every 10 minutes. Ensure your API key is up-to-date for accurate data retrieval.

### Can I customize the appearance of the weather information?

At the moment, you can customize the display name on the widget for the location. In later versions, customization options will be added for other parts of the widget.
